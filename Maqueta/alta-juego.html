Aquí puedes definir los diferentes grupos en los que se organizará la clasificación de tu juego. Para cada grupo debes definir una puntuación mínima y una puntuación máxima. Para el grupo de nivel superior, puedes definir sólo una puntuación mínima.

Si la puntuación de un juego puede depender de valoraciones adicionales, en este formulario puedes especificar las preguntas para que los participantes valoren la actuación del usuario en el juego a modo de encuesta.

Aquí puedes indicar la asignación de puntos en base a la media obtenida durante la evaluación del juego. Si, por ejemplo, la media de evaluación es 7, puedes especificar un intervalo de asignación de 5 puntos para una evaluación con una media entre 7 y 9. También puedes especificar valores negativos. Recuerda que la evaluación media se obtiene de la suma de las respuestas de todos los participantes.
